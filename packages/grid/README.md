# @coleski/datagrid

Ultra-fast canvas-based data grid for React.

> **Note**: Originally created by [Archer Hume](https://github.com/archerhume).

## Installation

```bash
npm install @coleski/datagrid
# or
bun add @coleski/datagrid
```

## Usage

```tsx
import { Grid } from '@coleski/datagrid'

function App() {
  return <Grid />
}
```
